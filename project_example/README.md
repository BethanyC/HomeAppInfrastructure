### How to run Home App from a Vagrang machine.

#Run the following commands:
vagrant up --provision
vagrant ssh
#You are now logged in to vagrant, in vagrant vm run commands:
sudo su home
cd /opt/home_app
java Menu
#You are now using the app, when finished run commands:
logout
vagrant destroy
